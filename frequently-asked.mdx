<AccordionGroup>
    <Accordion title="Is RestoreCord safe?" icon="shield-check">
        Yes, RestoreCord is safe. We're using [Discord OAuth2](https://discord.com/developers/docs/topics/oauth2) Feature.
        When you Verify with RestoreCord, you're redirected to Discord's OAuth2 page, where you Authorize the application to put you in the server.
    </Accordion>

    <Accordion title="Is RestoreCord free?" icon="dollar-sign">
        Yes, the basic features like Verification and Member Pulling are free. We do also have a [Premium](https://restr.co) and [Business](https://restr.co) subscription for more features.
    </Accordion>

    <Accordion title="How do I pull my members?" icon="download">
        Go to the [Servers](https://restr.co/servers) page, and click on the Migrate button on the server you want to pull members from.
        Then Select the server you want to pull members to, and click on the Pull button.
    </Accordion>

    <Accordion title="Does RestoreCord log my IP-Address?" icon="globe">
        The Server owners can enable or disable IP-Logging for their server. If IP-Logging is enabled, your IP-Address will be logged.
    </Accordion>

    <Accordion title="How do I turn the Bot on?" icon="toggle-on">
        The Bot works offline and doesn't need to be turned on. But If you really want the Bot to show as online, you can Purchase the [Business](https://restr.co/upgrade) subscription, and follow this [Guide](/guides/setup-commands).
    </Accordion>
</AccordionGroup>
