<Tip>First of all, we have to invite the bot to your Discord Server before creating the server on the Dashboard.</Tip>

> Headover to the Dashboard, select "Custom Bots" in the Navigation bar and click the invite button of the bot you are planning to use.

</Frame>
>

<Note>A Discord window should popup, select the corresponding server you wish to use RestoreCord on.</Note>
