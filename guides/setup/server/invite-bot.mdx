<Tip>First of all, we have to invite the bot to your Discord Server before creating the server on the Dashboard.</Tip>

> Headover to the Dashboard, select "Servers" in the Navigation bar and click the **CREATE NEW SERVER** button.

<Frame>
  <img src="/images/setup_create-server_select-bot.png" />
</Frame>
>

<Note>A Discord window should popup, select the corresponding server you wish to use RestoreCord on. Make sure to close the window after inviting the bot.</Note>

> Then click on **NEXT** to proceed to the next step.
